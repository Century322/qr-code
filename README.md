# QR code - 艺术二维码生成器

一个支持渐变色、自定义矢量图标拼接、独立定位眼设计的艺术二维码生成器。

## 功能特性

- 🎨 **渐变色支持** - 支持线性渐变和径向渐变
- ✨ **多种数据点样式** - 方形、圆形、圆角、菱形、星星、爱心、叶子
- 👁️ **独立定位眼设计** - 可自定义外框和内点样式
- 🎭 **迷彩背景** - 可选的随机迷彩背景效果
- 🌙 **深色模式** - 支持明暗主题切换
- 📱 **响应式设计** - 完美适配移动端和 PC 端
- 💾 **本地存储** - 自动保存用户设置

## 本地运行

**前置条件:** Node.js 18+

1. 安装依赖:
   ```bash
   npm install
   ```

2. 运行开发服务器:
   ```bash
   npm run dev
   ```

3. 构建生产版本:
   ```bash
   npm run build
   ```

## 技术栈

- React 19
- Vite 6
- Tailwind CSS 4
- TypeScript
- qrcode (二维码生成库)
